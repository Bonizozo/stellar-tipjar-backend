Quickstart

The Stellar Tipjar API lets you create creator profiles and record on-chain tips verified against the Stellar network.

Production: https://api.stellar-tipjar.com   |   Local: http://localhost:8000

Authentication

Obtain a JWT pair by registering or logging in, then pass the access token as a Bearer header:

Authorization: Bearer <access_token>

Access tokens expire after 15 minutes. Refresh via POST /api/v2/auth/refresh.

Code Samples

POST Register a creator

curl -X POST https://api.stellar-tipjar.com/api/v2/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "alice",
    "wallet_address": "GAAZI4TCR3TY5OJHCTJC2A4QSY6CJWJH5IAJTGKIN2ER7LBNVKOCCWN",
    "password": "supersecret123"
  }'
import requests

resp = requests.post(
    "https://api.stellar-tipjar.com/api/v2/auth/register",
    json={
        "username": "alice",
        "wallet_address": "GAAZI4TCR3TY5OJHCTJC2A4QSY6CJWJH5IAJTGKIN2ER7LBNVKOCCWN",
        "password": "supersecret123",
    }
)
tokens = resp.json()
access_token = tokens["access_token"]
const resp = await fetch("https://api.stellar-tipjar.com/api/v2/auth/register", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    username: "alice",
    wallet_address: "GAAZI4TCR3TY5OJHCTJC2A4QSY6CJWJH5IAJTGKIN2ER7LBNVKOCCWN",
    password: "supersecret123",
  }),
});
const { access_token, refresh_token } = await resp.json();
let resp = reqwest::Client::new()
    .post("https://api.stellar-tipjar.com/api/v2/auth/register")
    .json(&serde_json::json!({
        "username": "alice",
        "wallet_address": "GAAZI4TCR3TY5OJHCTJC2A4QSY6CJWJH5IAJTGKIN2ER7LBNVKOCCWN",
        "password": "supersecret123",

    .send().await?;
let tokens: serde_json::Value = resp.json().await?;

POST Record a tip

curl -X POST https://api.stellar-tipjar.com/api/v2/tips \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{
    "username": "alice",
    "amount": "10.5",
    "transaction_hash": "a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2",
    "message": "Keep up the great work!"
  }'
resp = requests.post(
    "https://api.stellar-tipjar.com/api/v2/tips",
    headers={"Authorization": f"Bearer {access_token}"},
    json={
        "username": "alice",
        "amount": "10.5",
        "transaction_hash": "a1b2c3d4...",
        "message": "Keep up the great work!",
    }
)
tip = resp.json()
const resp = await fetch("https://api.stellar-tipjar.com/api/v2/tips", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": `Bearer ${accessToken}`,
  },
  body: JSON.stringify({
    username: "alice", amount: "10.5",
    transaction_hash: "a1b2c3d4...",
    message: "Keep up the great work!",
  }),
});
const tip = await resp.json();
let resp = client
    .post("https://api.stellar-tipjar.com/api/v2/tips")
    .bearer_auth(&access_token)
    .json(&serde_json::json!({
        "username": "alice", "amount": "10.5",
        "transaction_hash": "a1b2c3d4...",
        "message": "Keep up the great work!",
    }))
    .send().await?;

GET List creator tips (paginated)

curl "https://api.stellar-tipjar.com/api/v2/creators/alice/tips?page=1&limit=20"
resp = requests.get(
    "https://api.stellar-tipjar.com/api/v2/creators/alice/tips",
    params={"page": 1, "limit": 20},
)
data = resp.json()  # { data, total, page, limit, total_pages, has_next, has_prev }
const resp = await fetch(
  "https://api.stellar-tipjar.com/api/v2/creators/alice/tips?page=1&limit=20"
);
const { data, total, has_next } = await resp.json();

SDK Generation

Generate a typed client from the OpenAPI spec using openapi-generator:

TypeScript

npx @openapitools/openapi-generator-cli generate \
  -i https://api.stellar-tipjar.com/docs/openapi.json \
  -g typescript-fetch \
  -o ./sdk/typescript \
  --additional-properties=typescriptThreePlus=true

Python

openapi-generator generate \
  -i https://api.stellar-tipjar.com/docs/openapi.json \
  -g python \
  -o ./sdk/python \
  --additional-properties=packageName=stellar_tipjar

Go

openapi-generator generate \
  -i https://api.stellar-tipjar.com/docs/openapi.json \
  -g go \
  -o ./sdk/go \
  --additional-properties=packageName=stellartipjar

Rate Limits

TierLimitApplies to
Read120 req/min per IPGET endpoints
Write30 req/min per IPPOST / PUT / DELETE

HTTP 429 Too Many Requests is returned when exceeded. Check the Retry-After header.

Error Format

All errors use a consistent JSON envelope:

{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Human-readable description",
    "details": {}
  }
